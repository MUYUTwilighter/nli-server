# MVP Order

1. Auth verification with no persisted login state.
2. Runtime instance creation that returns `presenceId` and `instanceToken`.
3. Friend graph CRUD.
4. Redis-backed Presence publish/query with TTL expiry.
5. WebSocket signaling relay between friends.
6. TURN credential endpoint.
7. Official friend list import.
8. Best-effort official deletion bridge.
9. Optional official add bridge if Mojang allows it again.
